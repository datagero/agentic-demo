# Strategic Review

Analyse all project context (RFP documents, ARCHITECTURE.md, open issues, current codebase) and produce a strategic assessment covering:

1. **RFP Alignment** — how well does our current plan address the RFP requirements?
2. **Prototype Scope** — is our demo scope right-sized for the 90-minute delivery window?
3. **Pitch Readiness** — do we have a clear problem statement, opportunity, concept, user journey, and demo plan?
4. **Risks & Gaps** — what could go wrong? What's missing?
5. **Recommended Next Actions** — prioritized list of what to do next.

Output the review as a structured markdown document saved to `.claude/strategic_plans/`.
