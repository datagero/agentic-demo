# Handoff Session

## Steps

1. **Check status** — `git status` and `gh issue list --state open`
2. **Push all branches** — ensure all work-in-progress branches are pushed
3. **Create handoff note** — summarize:
   - What was accomplished this session
   - What is in progress (with branch names)
   - What is blocked and why
   - Recommended next actions
4. **Update issues** — ensure all GitHub issues reflect current status
5. **Output the handoff note** for the next session to pick up
